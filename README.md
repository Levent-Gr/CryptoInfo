# Cypper

This is Cypper app.You can add the cryptocurrencies yo want to fallow closely to your favorites and control them with a single click.

## Libraries And Tools

- Used Language Kotlin
- MVVM Architecture
- Retrofit
- DI-Hilt
- Room Database
- Jetpack Navigation
- Coroutines
- View Binding
- Data Binding
- ViewModel
- Glide
- LiveData
- RecyclerView
- JUnit4
- Truth
- Hilt Testing
- Room Testing
- Coroutines Test
